# HireLens AI

AI-powered Resume Analyzer and Job Matching platform built using React, Tailwind CSS, Node.js, Express, and Gemini AI.

---

# Features

- Upload PDF and DOCX resumes
- AI-powered ATS Resume Analysis
- Resume Score Generation
- Skills Extraction
- Strengths & Weaknesses Detection
- AI Suggestions for Resume Improvement
- Job Description Matching
- Match Percentage Scoring
- Missing Skills Detection
- AI Recommendation Summary
- Downloadable PDF Report
- Responsive Premium UI
- Modern Dark Theme Design

---

# Technologies Used

- Mammoth

---

# How It Works

1. Upload a resume in PDF or DOCX format
2. Resume text gets extracted automatically
3. Gemini AI analyzes the resume
4. ATS score and insights are generated
5. User pastes a job description
6. AI compares resume with job requirements
7. Missing and matched skills are detected
8. AI generates job recommendation feedback

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/hirelens-ai.git
```

---

## Install Frontend Dependencies

```bash
npm install
```

---

## Install Backend Dependencies

```bash
cd backend
npm install
```

---

# Environment Variables

Create a `.env` file inside backend folder:

```env
GEMINI_API_KEY=your_api_key_here
```

---

# Run Frontend

```bash
npm run dev
```

---

# Run Backend

```bash
node server.js
```
