HireLens AI

An AI-powered Resume Analyzer and Job Matching web application built with React, Tailwind CSS, Node.js, Express, and Gemini AI.

Features

Upload PDF and DOCX resumes
AI-powered ATS resume analysis
Resume score generation
Skill extraction
Strengths and weaknesses detection
AI improvement suggestions
Job description matching
Match percentage scoring
Missing skills detection
AI job recommendation summary
Downloadable PDF analysis report
Responsive modern UI
Premium dark theme design

Technologies Used

Frontend
React.js
Tailwind CSS
React Circular Progressbar
jsPDF
Backend
Node.js
Express.js
Gemini AI API
Other Libraries
Mammoth
PDF Parse
CORS
dotenv

How It Works

User uploads a resume in PDF or DOCX format
Resume text is extracted and cleaned
Gemini AI analyzes the resume
ATS score, strengths, weaknesses, and suggestions are generated
User can paste a job description
AI compares resume with the job role
Matching skills and missing skills are displayed
AI generates a recommendation summary
